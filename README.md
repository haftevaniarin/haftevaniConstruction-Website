# Haftevani Construction Inc — Static Website

A modern, responsive, static website scaffold for a Los Angeles general contractor: Haftevani Construction Inc.

Includes original copy, responsive layout, and reusable components.

Files:
- `index.html` — Home page
- `about.html` — About Us
- `services.html` — Services overview
- `adu.html` — ADU & Garage Conversion
- `kitchen.html` — Kitchen Remodeling
- `bathroom.html` — Bathroom Remodeling
- `full-remodel.html` — Full Home Remodeling
- `new-construction.html` — New Construction
- `additions.html` — Home Additions
- `roofing.html` — Roofing / Exterior Renovation
- `projects.html` — Projects / Gallery
- `cities.html` — Cities We Serve
- `contact.html` — Contact page
- `assets/css/styles.css` — Global stylesheet
- `assets/js/site.js` — Small JS for mobile nav and form validation

How to view:
Open `index.html` in a browser. For local dev with live reload, use a static server such as `npx http-server`.
